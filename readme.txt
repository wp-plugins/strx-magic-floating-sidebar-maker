Magic Floating Sidebar Maker, makes your bog sidebar floatable.

== Description ==

If you have a classic column site structure, this plugin will make your sidebar follow your visitors while they scroll up and down the page.

Try it [Here](http://www.strx.it) or [Here](http://www.sulmonte.org)

Increase ads earnings making them always visible to the user, even when writing a comment on bottom of pages. Let visitors click on sidebar links without having to go up, at the beginning of the page

See also: [Official Blog Post](http://www.strx.it/2010/11/wordpress-floating-sidebar-plugin/)

== Installation ==

Upload the plugin to your blog, Activate it, add to the sidebar from the Widgets admin panel
